# API Productos

## 📌 Descripción
Sistema desarrollado con arquitectura en N-Capas usando Spring Boot.

## 🧱 Tecnologías
- Java
- Spring Boot
- PostgreSQL
- OpenAPI (Swagger)

## 📂 Estructura
- controller
- service
- repository
- entity
- dto
- config

## 🔄 Funcionalidades
- CRUD de productos
- Mapeo Entity ↔ DTO
