InvenEasy

Objetivo y funcionalidad
Este proyecto es una herramienta interna para la gestión, cálculo y seguimiento del inventario de una empresa de accesorios tecnológicos.
El sistema permite automatizar el proceso de venta de estos productos, aplicando un control riguroso del stock disponible. De esta forma, se obtienen cálculos precisos en tiempo real sobre la existencia de cada artículo en la empresa, facilitando la toma de decisiones y la planificación de compras y ventas.

Capacidad del sistema InvenEasy 
Productos: Registrar, listar, modificar datos y dar de baja artículos del inventario.
Entradas de inventario: Registrar compras o ingresos de productos, actualizando automáticamente el stock en tiempo real y generando movimientos de inventario.
Salidas / Ventas: Registrar ventas de accesorios tecnológicos, descontando unidades del stock y validando existencias mínimas antes de confirmar la operación.
Cálculo automático: Obtener balances en tiempo real sobre la existencia de cada articulo.
Ajustes de inventario: Modificar, corregir o ajustar cantidades de stock (por daños, errores o inventarios físicos), con actualización inmediata de saldos y trazabilidad de cambios.

Arquitectura del Sistema
/frontend: Interfaz para la visualización de productos
/backend: Lógica de cálculo de stock disponible 
/database: Tablas de productos, proveedores, ventas, inventario entre otros
